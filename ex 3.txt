Генерация смеси сигналов программным методом. Входные данные: файл (.ini), в котором указаны параметры сигналов, их тип, начальная и конечная их длительности.
Каждому  даётся два типа сигналов. Например: гармонический и ЛЧМ, или шум и кусок речи из бинарного файла
Цель: знакомство с парсингом файлов, применение настроек из файла, работа со сложными сигналами, сложная отладка. Знакомство BOOST(ил другим фрэймвороком или библиотекой для Парсинга .ini). 

Знакомство с паттерном фабрика. Абстрактная фабрика (для кого-то).

Ход работы:
1. Реализация абстракций для работы с файлом, основанных на лаб. Работе номер 2

2. Чтение конфигурационного файла и применение настроек в соответствии с uml-диаграммой

3. Парсер выдаёт список указателей на базовый конфиг. Один из них обязательно должен быть глобальным, остальные – это параметры генераторов

4. Генерация сигналов всех 
видов по их полиморфным параметрам

5. Склейка всех сигналов на одной тайм-линии (аддитивная смесь)

6. Формирование wav-заголовка

7. Сохранение в файл
Пример конфига: (частоты в герцах, время в мс) для типа гармонического + шум
